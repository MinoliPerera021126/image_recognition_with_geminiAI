# Image Recognition with Gemini AI

## Project Overview
This project demonstrates image recognition capabilities using Gemini AI in conjunction with Python and HTML components. The aim is to provide a user-friendly interface for uploading images and receiving detailed analysis and recognition results.

## Features
- **Image Upload**: Users can upload images through an HTML form.
- **Image Processing**: The backend, implemented in Python, processes the uploaded images using Gemini AI's recognition features.
- **Results Display**: Recognition results are displayed on the web interface in an informative manner.

## Technologies Used
- **Python**: For server-side logic and image processing.
- **Gemini AI**: For advanced image recognition capabilities.
- **HTML/CSS**: For frontend development and user interaction.

## Installation
1. Clone the repository:
   ```
   git clone https://github.com/MinoliPerera021126/image_recognition_with_geminiAI.git
   ```
2. Install required packages:
   ```
   pip install -r requirements.txt
   ```
3. Run the application:
   ```
   python app.py
   ```
4. Open your web browser and go to `http://localhost:5000`.

## Usage
- Upload an image using the provided form on the application page.
- View the recognition results immediately after processing.
